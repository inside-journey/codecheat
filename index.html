HTML

Tag Name Description
<table> Table It is used to create a table within the HTML Code
<tr> Table Row It is used to add a new horizontal row into the table
<td> table Data It is used to input new data into the cells
<table border=""> Table Border It is the old version of creating a border to improve readability in our table - however it is not used anymore regardless of the browser still reading. *Refer to CSS for the new version
<tbody> Table Body Sections the table data inside code to keep it more organised and easy to read
<thead> Table Head Sections off the heading of the table
<tfoot> Table Foot Bottom part of the table, which can be used for sums, differences or data results
